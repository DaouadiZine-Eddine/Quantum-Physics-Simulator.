# Quantum-Physics-Simulator.
The following project simulates the behavior of quantum particles when hitting a screen, following the principle of Young's experiments in order to offer an accurate representation. The project's language is french, and offers a highly detailled user interface that allows a deeper understanding of quantum phenomena.

# How to use it ?
In order to use the simulator, download the repository and execute the python file in your compiler.
You should be able to use the simulator this way, and access all of its options. The program creates a file for the figure made, saves it on your computer before opening it. Thus you can access it at any time you want.

# Motivation.
I love photonics, and quantum physics especially optics and thus I thought making such a project would be interesting from both a theoretical point of view but also as an exercice for physical modelisation.
Additionally, I believe that the simulator allows for a better understanding for physics students of quantum particles and their properties.
